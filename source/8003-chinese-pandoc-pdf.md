# Ubuntu上编译中文pdf

Excellent intro to latex <https://liam.page/2014/09/08/latex-introduction/>

```
wget https://github.com/jgm/pandoc/releases/download/2.18/pandoc-2.18-1-amd64.deb

apt install texlive-lang-chinese texlive-xetex
```
